# Function: isArray()

> **isArray**(`v`): `boolean`

Checks if the given value is an array.

## Parameters

• **v**: `unknown`

The value to check.

## Returns

`boolean`

- Returns true if the value is an array, otherwise false.

## Example

```ts
console.log(isArray([1, 2, 3])); // true
console.log(isArray({})); // false
```
