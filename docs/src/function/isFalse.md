# Function: isFalse()

> **isFalse**(`v`): `boolean`

Checks if the given value is false.

## Parameters

• **v**: `unknown`

The value to check.

## Returns

`boolean`

- Returns true if the value is false, otherwise false.

## Example

```ts
console.log(isFalse(false)); // true
console.log(isFalse(true)); // false
```
