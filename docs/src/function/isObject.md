# Function: isObject()

> **isObject**(`obj`): `boolean`

Checks if the given value is an object.

## Parameters

• **obj**: `unknown`

The value to check.

## Returns

`boolean`

- Returns true if the value is an object, otherwise false.

## Example

```ts
console.log(isObject({})); // true
console.log(isObject(null)); // false
```
