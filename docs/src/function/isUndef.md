# Function: isUndef()

> **isUndef**(`v`): `boolean`

Checks if the given value is undefined or null.

## Parameters

• **v**: `unknown`

The value to check.

## Returns

`boolean`

- Returns true if the value is undefined or null, otherwise false.

## Example

```ts
console.log(isUndef(undefined)); // true
console.log(isUndef(null)); // true
console.log(isUndef({})); // false
```
