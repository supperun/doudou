# Function: toNumber()

> **toNumber**(`val`): `number`

Converts the given value to a number.

## Parameters

• **val**: `any`

The value to convert.

## Returns

`number`

- Returns the number representation of the value or the original value if conversion fails.

## Example

```ts
console.log(toNumber('1')); // 1
console.log(toNumber('a')); // NaN
```
