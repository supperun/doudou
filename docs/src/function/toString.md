# Function: toString()

> **toString**(`val`): `String`

Converts the given value to a string representation.

## Parameters

• **val**: `any`

The value to convert.

## Returns

`String`

- Returns the string representation of the value.

## Example

```ts
console.log(toString([1, 2, 3])); // "[1, 2, 3]"
console.log(toString({})); // "{}"
```
